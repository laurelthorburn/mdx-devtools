# Getting Started

Welcome to the MDX DevTools demo! This page is rendered using MDX with several plugins.

  This callout is a custom component passed to MDX!
</Callout>

## Features

MDX allows you to use JSX in your markdown content. Here's what this demo showcases:

- **remark-gfm**: GitHub Flavored Markdown support
- **rehype-slug**: Automatic heading IDs
- **Custom components**: Callout, CodeBlock, etc.

## Code Example

```js
import mdxDevTools from '@mdx-devtools/vite-plugin';

export default defineConfig({
  plugins: [
    mdx(),
    mdxDevTools({
      remarkPlugins: ['remark-gfm'],
      rehypePlugins: ['rehype-slug'],
    }),
  ],
});
```
